Инструменты: HTML5 (localstorage..), javascript

1. Если сейчас ничего не проигрывается, то во всех вкладках показывается одинаковый playlist. Проигрывание можно начать с любой вкладки
Эта вкладка становится главной
2. Главная вкладка что-то играет. Во всех вкладках показывается текущий прогресс, элементы управления и остаток плейлиста.
3. При взаимодействии пользователя с главной вкладкой просто выполняются требуемые действия.
4. При взаимодействии пользователя с другой вкладкой, эта вкладка становится главной.
